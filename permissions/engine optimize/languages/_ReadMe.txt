When making a language file for the program, the thing to remember is
that the program was written and spaced around English.
So you will want to abbreviate as much as possible to try and keep the length 
of the translated words as long or shorter than the English part.
This way all text will still fit properly.


Open the .lng files in notepad, make sure to save them in Unicode format.

At the end of each line is /|\ this is used to tell the program where to stop,
this is used for text that is multiple lines.

On the left is the English text that is looked fro and then after then | is the translated text that will be used.
