## 💡 Advanced Usage

### Direct API Calls
```bash
# Get system info
curl -H "X-API-Key: dev_key_12345" \
  http://localhost:7860/system-info

# Run smart boost (dry-run)
curl -X POST \
  -H "X-API-Key: dev_key_12345" \
  "http://localhost:7860/smart-boost?dry_run=true"

# Execute specific tool
curl -X POST \
  -H "X-API-Key: dev_key_12345" \
  -H "Content-Type: application/json" \
  -d '{"tool_name":"clean_temp","dry_run":true}' \
  http://localhost:7860/execute-tool
```

### Custom Scripts
Use `windows_cmd_executor.py` for custom operations:
```bash
python windows_cmd_executor.py system_info
python windows_cmd_executor.py defrag --drive C
python windows_cmd_executor.py scan_disk --drive C --fix true
```

### Scheduled Automation
Create Windows Task Scheduler jobs to run tools automatically:
```batch
REM Run Smart Boost nightly at 2 AM
REM Add to Task Scheduler with:
python d:\NeoOptimize\backend\neoai_backend.py
```

## 📞 Support & Troubleshooting

### Quick Troubleshooting
1. Backend won't start? Check Python: `python --version`
2. UI can't connect? Check health: `curl http://localhost:7860/health`
3. Tool fails? Check logs: `type d:\NeoOptimize\logs\neoai_backend.log`
4. Need admin? Right-click terminal → Run as Administrator

### Log Files
- Backend: `d:\NeoOptimize\logs\neoai_backend.log`
- Commands: `d:\NeoOptimize\logs\windows_cmd_executor.log`
- Rotate monthly (automatic)

### Common Issues
- **"Permission Denied"** → Run as admin
- **"Connection Refused"** → Start backend first
- **"Port Already Used"** → Change port in code or kill process
- **"Module Not Found"** → `pip install -r requirements-neoai.txt`
