# Public Source Distribution

This repository contains the public NeoOptimize source distribution.

Included:

- Windows optimization and maintenance PowerShell modules.
- React/Tauri desktop UI source.
- Local Python AI engine source and tests.
- .NET launcher/utility source.
- Standalone public installer scripts.
- Package-manager manifests, checksums, screenshots, and public docs.

Excluded:

- Private remote-management backend and dashboard.
- Private endpoint connector service source and binaries.
- Lab infrastructure, VM/runtime reports, generated build output, dependency caches, release binaries, credentials, certificates, private keys, and local environment files.

The installer asset remains published through GitHub Releases. The repository
source tree is intentionally kept reviewable and rebuildable without private
backend infrastructure.
